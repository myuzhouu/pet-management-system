## 项目简介

这是一个轻量级的 Java 控制台小项目，现在被重构成了一个带有“节目感”的演说秀：

- 多位角色依次登台
- 每位角色拥有自己的称号、台词池和高光时刻
- 节目会动态累计气氛值
- 结尾会进入统一的终场环节

## 目录结构

- `src`：Java 源码
- `bin`：编译输出
- `lib`：依赖库

## 运行方式

先编译：

```bash
javac -encoding UTF-8 -d bin src/Test/*.java
```

再运行：

```bash
java -cp bin Test.App
```

## 当前类设计

- `App`：程序入口，负责组装节目和嘉宾
- `TalkShow`：节目流程控制，负责开场、回合、终场和气氛值
- `AiGuo`：角色行为接口
- `FamousMan`：具体嘉宾实现

## 可以继续玩的方向

- 加入用户输入，让观众选择下一位嘉宾
- 加入更多角色类型，而不只是一种 `FamousMan`
- 给每位角色增加不同的情绪值和台风风格
- 做成回合制小游戏或问答互动版
